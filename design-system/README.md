# Throughline dashboard — design system (current state)

This is the Throughline dashboard's current UI extracted as standalone design-system components, so the existing look can be reviewed and redesigned in claude.ai/design. Every component is a single openable HTML file with its CSS inlined (no external deps, no JS, no CDN), pulled verbatim from the `STYLE` constant and view builders in `packages/dashboard/views.mjs` — so this bundle reflects the product exactly as it renders today (the "before"). Each file begins with a `<!-- @dsCard group="..." -->` marker the Design System pane uses to build its index. Groups: **Foundations** (colors, typography, spacing/radius tokens), **Components** (tabs, section/card container, badges & pills, note/honesty callout, drill-down panel), **Data** (ledger bar + summary, standard data table), and **Visualizations** (the signature Flow value-stream diagram, the Connections reconciliation graph, and the single stage node).
